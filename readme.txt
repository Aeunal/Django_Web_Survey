You should follow the below steps to run the django project:

1. Step: Create a python environment and activate it. (python version: 3.6, 3.7, 3.8)

2. Step: Install requirements.txt

3. Step: Run the "Script_1.py" on "python manage.py shell".

4. Step: Run the "Script_2.py" on "python manage.py shell".

5. Step: Everything is ready. You can just write "python manage.py runserver" to runserver.
